# Practical Jobs-to-be-Done — Landing Page

The root landing page for [app.jtbd.one](https://app.jtbd.one).

A single self-contained index.html that showcases the app suite:

- Zero dependencies — pure HTML, CSS, vanilla JS
- Light + Dark theme via CSS custom properties and localStorage
- Hexagon tessellation hero background (SVG)
- Deployed on Vercel

## Development

Open index.html directly in any browser. No build step required.

## Deployment
